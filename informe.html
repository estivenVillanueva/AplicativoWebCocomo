Variables Implementadas

El modelo COCOMO 81 intermedio implementa las siguientes variables:

Factores de Ajuste

  • RELY - Fiabilidad Requerida
  • DATA - Tamaño de la Base de Datos
  • CPLX - Complejidad del Producto
  • TIME - Restricciones de Tiempo
  • STOR - Restricciones de Almacenamiento
  • VIRT - Volatilidad de la Máquina Virtual
  • TURN - Tiempo de Respuesta
  • ACAP - Capacidad de los Analistas
  • AEXP - Experiencia en la Aplicación
  • PCAP - Capacidad de los Programadores
  • VEXP - Experiencia en la Máquina Virtual
  • LEXP - Experiencia en el Lenguaje
  • MODP - Prácticas Modernas de Programación
  • TOOL - Uso de Herramientas de Software
  • SCED - Restricciones de Calendario

Modos de Proyecto

  • Orgánico - Proyectos pequeños y simples
  • Semi-Aislado - Proyectos de tamaño medio
  • Empotrado - Proyectos complejos con requisitos estrictos

Ejemplo de Cálculo

Para un proyecto de 50 KLOC en modo orgánico:

Datos de entrada:

  • Tamaño del Proyecto: 50 KLOC
  • Modo: Orgánico
  • Sueldo Mensual: $2000 USD
  • Factores de Ajuste: Valores nominales

Cálculos:

  • Esfuerzo = 3.2 * (50)^1.05 * 1.0 = 178.19 persona-mes
  • Duración = 2.5 * (178.19)^0.38 = 12.34 meses
  • Tamaño del Equipo = 178.19 / 12.34 = 14.44 programadores
  • Costo Total = $2000 * 14.44 * 12.34 = $356,379.20 USD

Interpretación de Resultados

Para un proyecto de 50 KLOC en modo orgánico, se estima que:

  • Se requerirán aproximadamente 178.19 persona-mes de esfuerzo
  • Con un equipo de 15 programadores, el proyecto tomará 12.34 meses
  • El costo total estimado es de $356,379.20 USD
  • Este es un proyecto de tamaño medio que requiere un equipo experimentado
